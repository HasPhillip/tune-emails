HOW TO GENERATE A NEW TEMPLATE:

Comment out the modules you don't need in the boilerplate.php file.

Change the theme colors in the styles.scss file.

Open the boilerplate.php file in a browser and copy the source. Save as a new html file.

Paste the content from the style.css file into a <style> block in the new html file.

Delete the <link> reference in the new html file.

Copy the source from the new html file. Inline the CSS: http://zurb.com/ink/inliner.php
